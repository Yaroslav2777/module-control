# AI-проhкт для ревʼю коду

Ви — AI-рецензент коду. Перегляньте наведену функцію JavaScript і визначте code smells, проблеми з читабельністю та стилем, не змінюючи поведінку. Надайте відповідь у структурованому списку з:

1. виявленими проблемами,
2. коротким поясненням для кожної,
3. відрефакторованою версією функції з використанням сучасного JavaScript (`const`/`let`, методи масивів, строгі порівняння, змістовні назви).

## Функція для ревʼю:

```js
function process(d) {
  var r = [];
  for (var i = 0; i < d.length; i++) {
    if (d[i].active == true) {
      if (d[i].age > 18) {
        r.push(d[i].name + " - " + d[i].age);
      }
    }
  }
  return r;
}
```
