# Завдання 4 — AI code review та рефакторинг

Оригінальна функція була переглянута та рефакторнa з використанням сучасного JavaScript.

## Файли
- `prompt.txt` — структурований prompt для AI code review
- `code-smells.txt` — знайдені проблеми
- `refactor.js` — відрефакторений код

## Зміст
- використано `const` / `let`
- застосовано `filter` і `map`
- уникнуто `var` і `==`
